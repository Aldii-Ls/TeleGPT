# AyGemuy Telegram Bot

# Installation

## Windows
* [`Download Node JS`](https://nodejs.org/en/download/)
* [`Download Git`](https://git-scm.com/download/win)


## Cloning this repo
```cmd
> git clone https://github.com/AyGemuy/TeleGPT
> cd TeleGPT
```

## Install the package
```cmd
> npm i
```

## Edit config file
Edit the required value in `config.json`. You can get the apikey at [`AyGemuy Rest API`](http://api-taylor.cyclic.sh/). And get bot token at [`@BotFather`](http://t.me/BotFather).
```json
{
    "apikey": "",
    "bot_token": "",
    "owner": "AyGemuy",
    "ownerLink": "https://t.me/wudysoft",
    "version": "1.0.0",
    "prefix": "/"
}
```

## Run the bot
```cmd
> npm start
```
